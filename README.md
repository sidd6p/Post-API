## Run Locally


- Clone the project

```bash
   https://github.com/sidd6p/Post-API.git
```

- Go to the Post-AP directory
```bash
   cd Post-AP

```
- Install requirements
```bash 
   pip install -r requirements.txt
```
- Run the API
```bash 
   uvicorn app.main:app 
```
